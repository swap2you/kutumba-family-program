# Rights and Use

## Repository posture

KUTUMBA is a **public documentation and curriculum-development repository**. Public access applies only to general program content, source-controlled operating models, curriculum materials, templates, and non-sensitive reference metadata. Private participant and operational records are **prohibited** from Git.

## KUTUMBA-authored original content

| Field | Value |
|---|---|
| Licence | [Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International (CC BY-NC-SA 4.0)](LICENSE.md) |
| Scope | Original documents authored for the KUTUMBA program and repository |
| Attribution | Credit KUTUMBA and link to this repository when sharing |
| Commercial use | Not permitted under CC BY-NC-SA without separate written permission |
| Derivatives | Must use the same licence |

See [LICENSE.md](LICENSE.md) for full terms.

## What the KUTUMBA licence does NOT cover

The CC BY-NC-SA 4.0 licence **does not** apply to and **does not** grant redistribution rights for:

| Category | Status in this repository |
|---|---|
| Śrīla Prabhupāda’s copyrighted books and purports | Cited only — BBT / authorised publisher rights |
| Bhaktivedanta Book Trust publications | Not relicensed |
| Third-party photographs, artwork, recordings, media | Rights remain with respective holders |
| Legacy Bhaktivriksha documents | Reference-only index — not canonical, not bulk-copied |
| Granth PDF catalog entries | Reference-only — not bulk-copied |
| Jayapataka Swami Maharaj Bhaktivriksha materials | Reference-only — rights review required before any use |
| Temple or ISKCON branding | Not KUTUMBA-owned |
| Sources with separate restrictions | Governed by original rights holder |

Do not relabel third-party content as KUTUMBA-owned.

## Permitted use of this repository

| Use | Allowed |
|---|---|
| Read and study public curriculum and governance docs | Yes |
| Fork for non-commercial ISKCON-family formation adaptation | Yes, under CC BY-NC-SA for KUTUMBA-authored parts |
| Bulk-copy legacy indexed PDFs from local paths into another repo | **No** |
| Publish participant private data | **No** |
| Present indexed legacy modules as approved KUTUMBA curriculum | **No** without formal adoption record |

## Adoption of legacy material

Before any legacy Bhaktivriksha, Granth, or Jayapataka BV content becomes canonical:

1. Cite source file ID and portable path
2. Complete rights disposition review
3. Obtain doctrinal, worship, and safety reviewer sign-off as applicable
4. Record modifications vs. original in the adoption log

See `17-reviews-and-audits/LEGACY-ADOPTION-REVIEW.md`.

## Quotation and scripture

- Scripture quotations must trace to authorised BBT/Prabhupāda sources
- Do not fabricate quotations or purports
- Mark unverified claims for research review

# Scripts

Maintenance and build scripts for the KUTUMBA repository.

| Script | Purpose |
|---|---|
| `ingest-sources.py` | Copy and hash current sources; index legacy references |
| `convert-docx-to-markdown.py` | Normalize DOCX to canonical Markdown |
| `split-curriculum-weeks.py` | Split first-six-month curriculum into weekly folders |
| `assemble-workstreams.py` | Generate scaffolding, legacy analysis, prompt library |
| `create-foundation.py` | Create repository foundation structure |
| `Validate-KutumbaRepository.ps1` | Run validation checks (exit 1 on critical failure) |

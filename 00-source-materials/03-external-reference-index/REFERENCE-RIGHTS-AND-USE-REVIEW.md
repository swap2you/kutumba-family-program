# Reference Rights and Use Review

## Policy

Legacy Bhaktivriksha, Granth, and Jayapataka Swami BV Material collections are **reference-only**. They are indexed in this repository but not bulk-copied into Git.

## Indexed collections

- LegacyBhaktivriksha: 1595 files — rights status: reference-only-rights-review-required
- Granth: 110 files — rights status: reference-only-rights-review-required

## Required human review

- Copyright and redistribution rights for any item proposed for adoption
- Worship authorization for any liturgical content
- Child-safety review for youth-facing legacy modules
- Doctrinal alignment review before adaptation

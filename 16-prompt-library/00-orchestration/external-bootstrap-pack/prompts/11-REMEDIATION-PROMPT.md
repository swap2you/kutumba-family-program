# Prompt 11 — Audit Remediation

Read the independent audit completely.

Create a remediation plan ordered by severity.

Fix only evidenced findings. Do not redesign unrelated content.

For each finding:

1. reproduce or verify it
2. identify root cause
3. make the smallest durable correction
4. update affected source/status/evidence records
5. run focused validation
6. run full validation
7. record the result
8. commit the correction

Create:

- `17-reviews-and-audits/AUDIT-REMEDIATION-REPORT.md`
- updated `CURRENT-STATUS.md`
- updated validation report

Do not close a finding merely because wording changed. Demonstrate the verification result.

Finish with a revised GO / GO WITH CONDITIONS / NO GO recommendation.

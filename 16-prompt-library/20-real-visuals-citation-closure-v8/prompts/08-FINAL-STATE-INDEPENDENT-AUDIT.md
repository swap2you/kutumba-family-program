# Prompt 08 — Final-State Independent Audit

## Objective

Perform the independent audit only after all V8 production commits are pushed or locally committed.

## Precondition

Record the production-complete HEAD.

The audit must not be authored before:

- real visual replacement;
- theology diagram correction;
- citation closure;
- media curation;
- Gamma packaging;
- validator completion.

## Fresh audit method

Use a fresh Cursor context when possible.

Read actual files, not summaries.

Deep-inspect:

- all 42 Cycle 1 SVGs;
- all 42 PNG derivatives;
- all module manifests;
- contact sheets;
- theology diagram;
- C1-W1 citation files;
- all C3-W2 files containing geography facts;
- every Cycle 1 Gamma asset map;
- one complete deck per audience;
- C1-W1–W6 media indexes;
- C3-W2 media index;
- all new validators.

## Audit tests

1. Open/render each visual.
2. Confirm it teaches what its class claims.
3. Confirm no metadata-card placeholders remain.
4. Confirm source and rights records.
5. Confirm canonical fact IDs.
6. Confirm Gamma card mapping.
7. Confirm no unauthorized art.
8. Confirm media is metadata-only.
9. Confirm audit commit order.
10. Confirm all human gates remain open.

## Report

Create:

`17-reviews-and-audits/V8-INDEPENDENT-REAL-VISUAL-CITATION-MEDIA-AUDIT.md`

Separate verdicts:

- repository integrity;
- citation closure;
- visual substance;
- Cycle 1 visual completeness;
- theology diagram;
- visual rights;
- media curation;
- Gamma upload readiness;
- kathā;
- internal pilot preparation;
- public publication.

Do not issue unconditional GO.

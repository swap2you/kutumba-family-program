## Objective

Complete the V7 media phase that remained partial.

## Required record fields

Each selected or deferred media item must include:

- media ID;
- module ID;
- exact title;
- speaker/creator;
- date;
- place;
- stable URL;
- source institution;
- media type;
- language;
- total duration if verified;
- recommended excerpt start/end if verified;
- transcript link;
- relevance;
- supported claim IDs;
- audience;
- age suitability;
- sensitive-content note;
- authority tier;
- rights posture;
- link status;
- doctrinal review;
- pedagogy review;
- selection status.

## Use the governed source library

Research from:

- VedaBase;
- Bhaktivedanta Archives;
- PrabhupadaVani;
- Vanisource/Vanipedia;
- official ISKCON Education;
- exact approved congregation sources.

Do not use a homepage when an exact lecture/resource page is available.

Do not download audio/video.

# Prompt 07 — Batch Enhance the Remaining 17 Modules

## Objective

Apply the approved C1-W2 pattern to the remaining modules without flattening their distinct theological purpose.

## Batch order

Process in small autonomous batches:

### Cycle 1
- C1-W1
- C1-W3
- C1-W4
- C1-W5
- C1-W6

### Cycle 2
- C2-W1 through C2-W6

### Cycle 3
- C3-W1 through C3-W6

Run validation after every module and a cycle audit after every cycle.

## Scope control

For each module create a “belongs here / belongs later / prerequisite” table.

Avoid repeating full body–soul material in every later lesson. Use links and brief recall.

## Minimum substantive outputs per module

- 6–15 verified Tier 1 references, adjusted to topic;
- direct stable links;
- 3 or more claim entries;
- 3 or more analogies or examples where appropriate;
- 3 or more contemporary applications;
- one misconceptions/boundaries file;
- substantive materials list;
- substantive assessment;
- substantive newcomer adaptation;
- Lāla–Lālī lesson;
- Kiśora–Kiśorī lesson;
- visual plan;
- at least one diagram source;
- three Gamma briefs;
- cycle project contribution;
- all review records.

Do not force irrelevant counts. A module may justify fewer analogies or more sources, but every omission must be explained.

## Contemporary issues

Use modern issues carefully:

- consumer identity;
- social media;
- body image;
- status anxiety;
- anger and speech;
- grief;
- free will and responsibility;
- karma victim-blaming;
- superstition;
- pseudoscience;
- guru misuse;
- sectarianism;
- performance culture;
- comparison;
- family pressure;
- screen habits;
- ecological care;
- animal compassion.

Do not name current scandals, devotees, temples, gurus, or factions unless the Program Director separately authorizes a formal case study.

## Cycle audits

Create:

- `17-reviews-and-audits/CYCLE-1-CONTENT-AUDIT.md`
- `17-reviews-and-audits/CYCLE-2-CONTENT-AUDIT.md`
- `17-reviews-and-audits/CYCLE-3-CONTENT-AUDIT.md`

Each cycle audit must confirm:

- sequence coherence;

- prerequisites;
- child progression;

- project readiness;
- celebration readiness;
- review gates.

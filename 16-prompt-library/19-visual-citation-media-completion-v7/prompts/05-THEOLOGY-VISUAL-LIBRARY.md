# Prompt 05 — Theology Visual Reference Library

## Objective

Create a review-controlled reference library for complex Gauḍīya Vaiṣṇava theology. This is not an instruction to teach every diagram in the first six months.

Create:

```text
14-research-source-register/theology-visual-library/
├── README.md
├── THEOLOGY-VISUAL-BACKLOG.yaml
├── THEOLOGY-FACT-REGISTRY.yaml
├── REVIEW-BOUNDARIES.md
├── krishna-and-expansions/
├── purusa-avataras/
├── avatara-categories/
├── bhagavan-paramatma-brahman/
├── energies-and-jiva/
├── creation-overview/
└── universal-form-boundaries/
```

## Priority diagrams

### 1. Kṛṣṇa and expansions

Prepare a diagram specification for the commonly presented Gauḍīya sequence:

Kṛṣṇa → Balarāma → first catur-vyūha → Nārāyaṇa → second catur-vyūha → Mahā-Viṣṇu → Garbhodakaśāyī Viṣṇu → Kṣīrodakaśāyī Viṣṇu.

Requirements:

- verify every node and relationship against Śrīla Prabhupāda’s books and exact VedaBase passages;
- distinguish direct source statement from synthesized teaching diagram;
- do not imply a chronological material process;
- include a review note where terminology varies by text/context;
- no publication approval without named doctrinal review.

Candidate source areas to verify:

- Caitanya-caritāmṛta Ādi-līlā Chapters 2 and 5;
- Śrīmad-Bhāgavatam 1.3;
- Śrīmad-Bhāgavatam 2.6;
- Brahma-saṁhitā 5.35, 5.39 and 5.48;
- Bhagavad-gītā 10.42;
- Śrīmad-Bhāgavatam 1.3.28.

Do not assume these references support every edge without checking context.

### 2. Three puruṣa expansions

Show functions distinctly:

- causal universes;
- entry into each universe;
- entry into every being/atom as Paramātmā.

Use source-exact labels and a synthesis disclaimer.

### 3. Avatāra category map

Create a controlled map of major categories only after verifying terminology:

- puruṣa;
- guṇa;
- līlā;
- manvantara;
- yuga;
- śaktyāveśa.

Do not overload child modules.

### 4. Bhagavān, Paramātmā and Brahman

Show one Absolute Truth realized in three features without collapsing personal distinctions or creating a simplistic “levels of God” graphic.

### 5. Energies and jīva

Differentiate:

- internal;
- marginal;
- external energies;

with exact source anchors and boundaries.

### 6. Creation overview

Create a high-level reference diagram, not a speculative scientific model.

### 7. Universal form boundaries

Explain that the universal form is a divinely revealed vision and not the final personal form of Kṛṣṇa. Avoid sensational imagery.

## Audience variants

For every approved reference diagram prepare specifications for:

- parent/facilitator;
- Kiśora–Kiśorī;
- Lāla–Lālī only when age-appropriate.

Many advanced diagrams should be `reference-only` for facilitators during Year 1.

## Asset method

Prefer original geometric SVG diagrams with text and icons. Do not copy classical paintings. Do not generate depictions of sacred personalities merely to fill space.

## Deliverables

- source-anchored diagram specifications;
- initial original SVG diagrams for the highest-priority items;
- source and rights registers;
- review worksheets;
- `build-evidence/V7-THEOLOGY-VISUAL-LIBRARY-REPORT.md`.

All assets remain `human-doctrinal-review-required`.

# Official Research Notes for V7

These notes guide Cursor’s research. They are not automatic permission to copy content.

## Gamma

Official Gamma pages state that Gamma can:

- start from an idea, pasted outline or imported content;
- generate or rework images;
- use themes/branding;
- export to PPT, PDF, PNG and Google Slides.

Use Gamma as a rendering/editor tool. Do not let Gamma invent doctrinal content.

Official entry point:

- https://gamma.app/

## GitHub and diagram assets

GitHub Markdown supports Mermaid diagrams. Mermaid remains useful for flows and maps, but Mermaid presence alone does not prove a meaningful visual library.

Git LFS stores pointers to large files. Do not introduce it until there is a documented requirement for large, rights-cleared media.

Official references:

- https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/creating-diagrams
- https://docs.github.com/en/repositories/working-with-files/managing-large-files/about-git-large-file-storage

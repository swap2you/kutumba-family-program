# Rights Decision Matrix

| Content | Default repository treatment | Escalation |
|---|---|---|
| KUTUMBA-authored summary | Commit with provenance | editorial review |
| Third-party webpage metadata | Commit URL/title/owner/date | none unless unclear |
| Short quotation | Minimal, necessary, attributed | citation + rights review |
| Full purport/synonyms/chapter | Do not commit | permission required |
| PDF workbook or training manual | Link and metadata only | permission/open licence |
| Audio/video | Metadata, duration, timestamp, link | permission for redistribution |
| Artwork/painting/photo | External link or placeholder | rights clearance |
| Auto transcript | Do not publish as authoritative | accuracy and rights review |

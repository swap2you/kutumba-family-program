# Source Tier Mapping

| Layer | Examples | Normal use | Not allowed by default |
|---|---|---|---|
| Tier A: controlling/provenance | VedaBase, Bhaktivedanta Archives, BBT/BBTi, official ISKCON/GBC/ministry | doctrine, provenance, policy, rights | bulk copying |
| Tier B: structured secondary | PrabhupadaVani, Vanisource, Vaniquotes, Vanipedia | discovery, topic mapping, alternate access | final claim without context check |
| Tier C: convenience/discovery | Desire Tree, PrabhupadaBooks, YouTube | media discovery, edition comparison | canonical status by assumption |
| Supplementary | PureBhakti, Harikatha, Gita Press | attributed background, narrative or edition comparison after review | overriding Prabhupāda-first conclusions |

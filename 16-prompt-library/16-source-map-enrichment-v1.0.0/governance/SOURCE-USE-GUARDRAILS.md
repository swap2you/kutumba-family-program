# Source Use Guardrails

## Core rule

The source map is a research directory, not a blanket licence.

## Preferred workflow

1. Discover from the supplied source map.
2. Verify ownership and exact page.
3. Select canonical provenance anchor.
4. Record alternate/mirror links.
5. Store metadata and original KUTUMBA summary.
6. Add a short excerpt only when necessary and reviewed.
7. Map to a claim, activity or media purpose.
8. Obtain human review before publication.

## Prohibited shortcuts

- bulk scrape;
- copy full books or purports;
- download and commit audio/video;
- copy BBT paintings;
- use an AI summary as authority;
- infer teacher approval;
- mix translations without edition labels;
- treat a mirror as canonical;
- treat an HTTP 200 response as rights clearance;
- treat source count as quality.

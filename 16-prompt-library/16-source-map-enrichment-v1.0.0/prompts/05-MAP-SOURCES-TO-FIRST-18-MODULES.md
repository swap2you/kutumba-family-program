# Prompt 05 — Map Sources to the First 18 Modules

## Goal

Create a source-coverage plan for every first-six-month module before adding content.

## Output

Create:

- `14-research-source-register/public-source-catalog/SOURCE-TO-MODULE-MAP.yaml`
- `build-evidence/MODULE-SOURCE-COVERAGE-REPORT.md`
- one `research/SOURCE-EXPANSION-BRIEF.md` inside each of the 18 module folders

## Source roles

For each module identify:

1. controlling Prabhupāda book references;
2. verified Prabhupāda lecture/conversation/letter/morning-walk candidates;
3. source-grounded kathā references;
4. official education/pedagogy resources;
5. child-resource candidates;
6. media candidates;
7. supplementary Gauḍīya/Sanātana references;
8. sources intentionally excluded;
9. rights status;
10. review required.

## Requirements

- No module should receive every available source.
- Select sources by teaching function.
- Prefer exact verse/chapter/event links over site homepages.
- Avoid using secondary quote pages when the full-context primary source is available.
- Do not use supplementary sources to manufacture doctrinal novelty.
- Every source recommendation must state:
  - why it belongs;
  - what claim or activity it supports;
  - audience;
  - risk of misuse;
  - rights posture.
- Record source conflicts or edition differences rather than silently harmonizing them.

## Coverage scoring

Score separately:

- primary text coverage;
- Prabhupāda spoken-source coverage;
- kathā coverage;
- parent application coverage;
- Lāla–Lālī coverage;
- Kiśora–Kiśorī coverage;
- visual/media coverage;
- rights readiness;
- human review readiness.

Do not convert this score into publication approval.

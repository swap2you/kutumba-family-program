# Prompt 08 — Enrich Supplementary Gauḍīya and Sanātana References

## Goal

Build a controlled supplementary reference layer for wider Gauḍīya and Sanātana materials without weakening the Prabhupāda-first source hierarchy.

## Sources named in the supplied map

- PureBhakti
- Harikatha
- Gita Press
- exact user-approved additional sources

## Governance

Classify all entries as one of:

- `recognized-gaudiya-supplementary`
- `sanatana-narrative-supplementary`
- `edition-comparison-only`
- `media-discovery-only`
- `pending-doctrinal-review`

## Rules

- Exact attribution is mandatory.
- Identify translator, editor, edition and publisher.
- Do not present a source as ISKCON-controlling merely because it is devotional.
- Do not merge translations from different editions.
- Do not quote or paraphrase controversial positions without context and human review.
- Use supplementary sources mainly for:
  - narrative background;
  - historical context;
  - devotional literature discovery;
  - terminology comparison;
  - activities or stories after doctrinal review.
- Prabhupāda's books remain the normal controlling teaching layer.

## Outputs

Create:

- `14-research-source-register/SUPPLEMENTARY-SOURCE-BOUNDARY-POLICY.md`
- `14-research-source-register/public-source-catalog/SUPPLEMENTARY-GAUDIYA-SANATANA.yaml`
- `build-evidence/SUPPLEMENTARY-SOURCE-BOUNDARY-REPORT.md`

## Module mapping

Add supplementary candidates to module expansion briefs only when:

- directly relevant;
- exact source known;
- no contradiction is hidden;
- reason for inclusion stated;
- human review marked.

# Prompt 07 — Enrich Education, Bhakti-vṛkṣa, and Child Resources

## Goal

Use official ISKCON education and congregation resources to improve pedagogy, activity design, facilitator preparation, child differentiation and family practice without copying third-party curriculum wholesale.

## Priority sources

- ISKCON Ministry of Education
- official Ministry media library/materials/books-for-kids/philosophy pages
- verified Sunday School resource pages and teacher guides
- ISKCON Congregational Development / Bhakti-vṛkṣa training resources
- user-supplied legacy Bhakti-vṛkṣa indexes as comparison references

## Extract only

- title and source metadata;
- intended age range;
- learning method;
- activity pattern;
- teacher-preparation pattern;
- assessment pattern;
- material needs;
- safety considerations;
- original KUTUMBA adaptation notes.

## Do not

- copy full workbooks or teacher guides;
- silently import another program's theology or sequence;
- replace the KUTUMBA three-year architecture;
- restore the old three-band child model;
- treat Bhakti-vṛkṣa as the KUTUMBA operating model.

## Apply to modules

For each module:

- improve Lāla–Lālī activities;
- improve Kiśora–Kiśorī activities;
- improve facilitator guidance;
- improve materials and assessment;
- improve newcomer adaptation;
- cite the educational inspiration source;
- label the final activity as KUTUMBA-authored adaptation.

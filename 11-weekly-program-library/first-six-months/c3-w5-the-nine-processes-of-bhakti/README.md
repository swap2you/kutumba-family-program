# C3-W5 — The Nine Processes of Bhakti

## Purpose

Give families a broad, scriptural map of devotional service so bhakti is experienced as a whole life of hearing, chanting, remembering, worship, prayer, service, friendship and surrender—not only classroom attendance.

## Cycle

Cycle 3 — first six months (Year 1 foundation)

## Key verse

Śrīmad-Bhāgavatam 7.5.23–24 — śravaṇaṁ kīrtanaṁ viṣṇoḥ smaraṇaṁ pāda-sevanam

## Minimum home practice

Continue the family’s core practice and add one underused process once.

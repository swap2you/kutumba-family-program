# C2-W3 — Birth, Death and Reincarnation

## Purpose

Teach transmigration as the soul’s movement through changing bodies under karma and consciousness, in a calm and age-appropriate way that supports responsibility and spiritual hope rather than fear or speculation.

## Cycle

Cycle 2 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 2.22 — vāsāṁsi jīrṇāni yathā vihāya

## Minimum home practice

Complete the one-minute bedtime remembrance three nights.

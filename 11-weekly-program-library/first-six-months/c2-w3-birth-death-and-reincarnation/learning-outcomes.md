## Learning Outcomes

| **Outcome domain** | **By the end of the session, participants should be able to…** |
| --- | --- |
| **Knowledge** | Explain reincarnation using BG 2.13 and the clothes analogy of BG 2.22. |
| **Skill** | Answer a basic question about death with truth, calmness and developmental sensitivity. |
| **Practice** | End the day with one minute of Kṛṣṇa remembrance rather than uncontrolled media. |
| **Character** | Sobriety, hope, compassion and reduced denial of mortality. |
| **Child** | Understand that the soul continues when the body stops working. |

## Primary Śrīla Prabhupāda Sources

- Bhagavad-gītā 2.13

- Bhagavad-gītā 2.20 and 2.22

- Bhagavad-gītā 2.27

- Bhagavad-gītā 8.5–6

- Bhagavad-gītā 14.14–15, introduced only at recognition level

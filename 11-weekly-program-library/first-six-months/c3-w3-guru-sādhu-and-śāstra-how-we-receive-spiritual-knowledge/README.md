# C3-W3 — Guru, Sādhu and Śāstra: How We Receive Spiritual Knowledge

## Purpose

Teach a safe and authoritative knowledge pathway through bona fide guru, saintly devotees and scripture, while clarifying roles, inquiry, service, verification and the limits of KUTUMBA authority.

## Cycle

Cycle 3 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 4.34 — tad viddhi praṇipātena paripraśnena sevayā

## Minimum home practice

Write one sincere spiritual question and locate one relevant Śrīla Prabhupāda source.

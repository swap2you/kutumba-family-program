# C1-W5 — The Temporary World and the Search for Permanent Happiness

## Purpose

Help families honestly recognize the limits of material arrangements, distinguish healthy enjoyment from the demand that temporary things provide permanent fulfillment, and redirect desire toward Kṛṣṇa-centered happiness.

## Cycle

Cycle 1 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 8.15 — duḥkhālayam aśāśvatam

## Minimum home practice

Before one enjoyable activity, pause for gratitude and three mahā-mantras.

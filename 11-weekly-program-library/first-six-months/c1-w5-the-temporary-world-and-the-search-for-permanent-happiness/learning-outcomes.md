## Learning Outcomes

| **Outcome domain** | **By the end of the session, participants should be able to…** |
| --- | --- |
| **Knowledge** | Explain aśāśvatam as temporary and connect it with recurring change, separation and frustration. |
| **Skill** | Examine an expectation and identify whether a temporary object is being asked to provide permanent security or identity. |
| **Practice** | Offer one enjoyable family activity to Kṛṣṇa by adding remembrance, gratitude or service. |
| **Character** | Contentment, realism, gratitude and reduced entitlement. |
| **Child** | Recognize that toys, treats and winning can be enjoyable but do not stay forever. |

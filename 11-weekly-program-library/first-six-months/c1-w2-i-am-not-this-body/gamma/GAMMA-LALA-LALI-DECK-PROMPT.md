# Gamma Prompt — C1-W2 Lāla–Lālī Deck

## Deck identity

- **Module:** C1-W2
- **Audience:** Lāla–Lālī ages 4–8 (core 4–6 + extension 7–8)
- **Purpose:** Same person through changing bodies; care for the instrument
- **Format:** 16:9, large text, minimal words per card
- **Recommended card count:** 10
- **Style:** Bright, simple, story-led
- **Theme:** Pictures, costumes, and kindness to bodies

## Global instructions

Large fonts. No death imagery. No frightening transitions. Paste card content exactly.

# C1-W2 — I Am Not This Body

## Two-minute summary

This week teaches one foundational distinction: the **body changes** (childhood, youth, old age) while the **conscious self continues**. Participants learn precise language — "I have a body" — and practice caring for the body as Kṛṣṇa's instrument rather than idolizing or despising it. Key anchor: [Bhagavad-gītā 2.13](https://vedabase.io/en/library/bg/2/13/). This module does **not** teach full soul ontology, transmigration, or karma — those come in later weeks (especially C1-W3).

**Memory line:** The body changes from childhood to old age, but the conscious self continues.

**Minimum home practice:** Identity pause once daily at a chosen trigger.

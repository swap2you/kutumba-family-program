# C1-W2 — I Am Not This Body

## Purpose

Help adults and children distinguish the changing body from the continuing conscious self, while learning to care for the body as Kṛṣṇa’s instrument rather than neglect or despise it.

## Cycle

Cycle 1 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 2.13 — dehino ’smin yathā dehe

## Minimum home practice

Use the identity pause once daily at a chosen trigger.

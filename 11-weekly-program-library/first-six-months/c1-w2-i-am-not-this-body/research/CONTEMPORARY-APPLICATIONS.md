# C1-W2 Contemporary Applications — Anonymized Case Studies

Eight cases for parent and youth tracks. All names and details are fictional composites.

---

## CS-01 — Social-media appearance pressure

| Field | Detail |
| --- | --- |
| **Presenting situation** | A 13-year-old spends hours editing photos before posting. Mood crashes when a post gets few likes. Says, "This is who I am." |
| **Mistaken conclusion** | Online image = full self; worth equals engagement metrics |
| **Source-grounded correction** | Body/appearance changes; the conscious self continues ([BG 2.13](https://vedabase.io/en/library/bg/2/13/)). Bodily identification intensifies suffering ([SB 10.84.13](https://vedabase.io/en/library/sb/10/84/13/)) |
| **Compassionate response** | Validate feelings first. Discuss screen limits as body-care. Rewrite identity: servant of Kṛṣṇa using body for real connection |
| **Facilitator caution** | No public confession of accounts; no shaming selfies |
| **Lāla–Lālī** | "Profile picture is like a costume — you are still you inside." Simple device/user analogy avoided; use costume relay |
| **Kiśora–Kiśorī** | Full case discussion; device/user analogy; reflective writing optional |

## CS-02 — Parent fearing aging

| Field | Detail |
| --- | --- |
| **Presenting situation** | A 48-year-old parent avoids family photos, says, "I'm not the person I used to be." Withdraws from community service |
| **Mistaken conclusion** | Changing body = loss of self-worth and purpose |
| **Source-grounded correction** | Same dehī through stages ([BG 2.13](https://vedabase.io/en/library/bg/2/13/)); constitutional service identity unchanged ([CC Madhya 20.108](https://vedabase.io/en/library/cc/madhya/20/108/)) |
| **Compassionate response** | Honor grief about aging. Reframe service: "What can this body do for Kṛṣṇa today?" |
| **Facilitator caution** | Do not mock vanity; watch for depression signals |
| **Lāla–Lālī** | Child draws parent at two ages; "Same Amma/Papa — different picture" |
| **Kiśora–Kiśorī** | Interview parent (optional) about what stayed same over 20 years |

## CS-03 — Child mocking disability

| Field | Detail |
| --- | --- |
| **Presenting situation** | A 10-year-old imitates a classmate's walk for laughs. Says at temple, "They're weird." |
| **Mistaken conclusion** | Body difference = less person; mockery harmless |
| **Source-grounded correction** | Soul not destroyed by bodily condition ([BG 2.20](https://vedabase.io/en/library/bg/2/20/)); mockery is violence against Kṛṣṇa's part-and-parcel |
| **Compassionate response** | Firm correction. Practice respectful language. Include, don't stare |
| **Facilitator caution** | Do not single out disabled child in group; use fictional scenario |
| **Lāla–Lālī** | Scenario card: friend with crutches — help, don't laugh |
| **Kiśora–Kiśorī** | Disability dignity segment; debate on boundaries |

## CS-04 — Devotee neglecting sleep/medicine

| Field | Detail |
| --- | --- |
| **Presenting situation** | A young adult skips prescribed medication and sleeps 4 hours, citing "I'm not this body — only chanting matters." |
| **Mistaken conclusion** | Spiritual slogan cancels medical and bodily care |
| **Source-grounded correction** | Body is instrument for service; neglect harms capacity to serve ([BG 2.16](https://vedabase.io/en/library/bg/2/16/) — proper discrimination) |
| **Compassionate response** | Private talk with mentor. Medicine + sleep as yukta-vairāgya |
| **Facilitator caution** | Medical advice only from qualified professionals; facilitator encourages compliance |
| **Lāla–Lālī** | Body-care sorting: sleep and medicine cards in "good for my body" |
| **Kiśora–Kiśorī** | Debate false motion: boundaries don't matter — refute with instrument care |

## CS-05 — Professional status collapse

| Field | Detail |
| --- | --- |
| **Presenting situation** | A parent loses a job title. Says, "I am nothing now." Stops attending programs |
| **Mistaken conclusion** | Career label = self ([SB 10.84.13](https://vedabase.io/en/library/sb/10/84/13/) — bodily/worldly identification) |
| **Source-grounded correction** | Identity is eternal servant of Kṛṣṇa ([CC Madhya 20.108](https://vedabase.io/en/library/cc/madhya/20/108/)); occupation is temporary role |
| **Compassionate response** | Acknowledge real financial stress. Separate role from worth. Small service step |
| **Facilitator caution** | No toxic positivity; connect to practical support if needed |
| **Lāla–Lālī** | Not primary case — parent track only |
| **Kiśora–Kiśorī** | Optional: "If I lost my team/status, who would I still be?" private writing |

## CS-06 — Prejudice based on body (race/ethnicity)

| Field | Detail |
| --- | --- |
| **Presenting situation** | Overheard comment that certain bodies are "less pure" or "less capable of bhakti." |
| **Mistaken conclusion** | Bodily birth label determines spiritual worth |
| **Source-grounded correction** | Soul is beyond bodily designation; vaiṣṇava transcends bodily labels (teach humbly without over-quoting) |
| **Compassionate response** | Interrupt prejudice. Affirm equal spiritual potential; practical inclusion |
| **Facilitator caution** | High safeguarding; may need leadership escalation |
| **Lāla–Lālī** | "Every body has a person inside who belongs to Kṛṣṇa." |
| **Kiśora–Kiśorī** | Structured discussion on respect; no debate on human worth |

## CS-07 — Cruelty toward animals or plants

| Field | Detail |
| --- | --- |
| **Presenting situation** | Children pull leaves off a temple plant "for fun"; teen says, "They're just matter." |
| **Mistaken conclusion** | "Not the body" extends to no compassion for other embodied beings |

## CS-08 — Bereaved family, insensitive teaching

| Field | Detail |
| --- | --- |
| **Presenting situation** | After a relative's death, a well-meaning visitor says, "Don't cry — they were just a body." Family feels hurt and angry |
| **Mistaken conclusion** | Philosophy replaces compassion; grief is "māyā" |
| **Source-grounded correction** | Distinction between body and soul does not forbid grief; sober knowledge ([BG 2.13](https://vedabase.io/en/library/bg/2/13/)) ≠ cold dismissal |
| **Compassionate response** | Listen first. Offer practical support. Philosophy only when invited |
| **Facilitator caution** | No death mechanics with children; bereaved adults — refer to pastoral care |
| **Lāla–Lālī** | Do not use this case in group; facilitator awareness only |
| **Kiśora–Kiśorī** | Sensitivity training: "What not to say to grieving friends" |

## CS-09 — Gendered safety (extension case)

| Field | Detail |
| --- | --- |
| **Presenting situation** | A teen is told "modesty doesn't matter — body is illusion" while facing harassment |
| **Mistaken conclusion** | Philosophy removes boundaries and safety planning |
| **Source-grounded correction** | Body is real enough to protect; boundaries matter; soul responsible for ethical action |

| **Facilitator caution** | Mandatory safeguarding escalation; not a debate topic |
| **Lāla–Lālī** | Body belongs to Kṛṣṇa — we protect it; tell a trusted adult if scared |
| **Kiśora–Kiśorī** | Boundaries debate concludes: harm matters |

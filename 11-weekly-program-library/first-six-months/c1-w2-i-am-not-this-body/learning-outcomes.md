## Learning Outcomes

| **Outcome domain** | **By the end of the session, participants should be able to…** |
| --- | --- |
| **Knowledge** | Explain the body–self distinction using Bhagavad-gītā 2.13. |
| **Skill** | Use precise language: ‘I have a body’ rather than treating the body as the complete self. |
| **Practice** | Pause once daily and remember: ‘I am Kṛṣṇa’s eternal servant; this body is for service.’ |
| **Character** | Respect, sobriety, reduced vanity and reduced contempt toward self or others. |
| **Child** | Recognize that the same person remains through baby, child and older-body stages. |

## Primary Śrīla Prabhupāda Sources

- Bhagavad-gītā 2.13

- Bhagavad-gītā 2.16–18

- Bhagavad-gītā 13.1–3

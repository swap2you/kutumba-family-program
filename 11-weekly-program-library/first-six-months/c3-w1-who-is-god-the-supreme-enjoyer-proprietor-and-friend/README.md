# C3-W1 — Who Is God? The Supreme Enjoyer, Proprietor and Friend

## Purpose

Give families a clear personal conception of God as the supreme conscious controller, proprietor, enjoyer, witness and well-wishing friend, and show how this knowledge brings practical peace.

## Cycle

Cycle 3 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 5.29 — bhoktāraṁ yajña-tapasāṁ ... suhṛdaṁ sarva-bhūtānām

## Minimum home practice

Before one daily duty, remember: Kṛṣṇa owns, Kṛṣṇa enjoys, Kṛṣṇa is friend.

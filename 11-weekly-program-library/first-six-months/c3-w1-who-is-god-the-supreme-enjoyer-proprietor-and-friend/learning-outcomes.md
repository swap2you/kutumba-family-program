## Learning Outcomes

| **Outcome domain** | **By the end of the session, participants should be able to…** |
| --- | --- |
| **Knowledge** | State the three peace-giving features in BG 5.29: supreme enjoyer, proprietor and friend of all beings. |
| **Skill** | Reframe one conflict over control, credit or possession using the peace formula. |
| **Practice** | Use one daily ‘Kṛṣṇa owns, Kṛṣṇa enjoys, Kṛṣṇa is friend’ remembrance before a family duty. |
| **Character** | Stewardship, gratitude, reduced possessiveness and trust. |
| **Child** | Say that Kṛṣṇa owns everything, accepts loving offerings and is the friend of every living being. |

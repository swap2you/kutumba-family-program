## Learning Outcomes

| **Outcome domain** | **By the end of the session, participants should be able to…** |
| --- | --- |
| **Knowledge** | Explain that material nature, prior conditioning, the body and other agents limit control. |
| **Skill** | Separate what is controlled, influenced and not controlled in a difficult situation. |
| **Practice** | Apply the three-step choice pause to one predictable trigger each day. |
| **Character** | Responsibility, courage, patience and willingness to seek guidance. |
| **Child** | Understand that feelings can be strong while choices still matter. |

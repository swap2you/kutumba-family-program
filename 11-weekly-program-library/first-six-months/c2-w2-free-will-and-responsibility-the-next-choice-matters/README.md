# C2-W2 — Free Will and Responsibility: The Next Choice Matters

## Purpose

Teach that the conditioned soul does not control every circumstance but retains meaningful, minute choice and responsibility, especially in how one responds to guidance, habits and present conditions.

## Cycle

Cycle 2 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 18.63 — vimṛśyaitad aśeṣeṇa yathecchasi tathā kuru

## Minimum home practice

Use pause–remember–choose once daily on the chosen trigger.

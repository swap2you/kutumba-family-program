## CS-01 — Angry group message

| Field | Detail |
| --- | --- |
| **Presenting situation** | A parent posts a sharp correction in the family chat at midnight. |
| **Mistaken conclusion** | Words are harmless if intention was good |

| **Facilitator caution** | No re-enacting real family conflicts |
| **Lāla–Lālī** | Action echo game — kind vs harsh words |

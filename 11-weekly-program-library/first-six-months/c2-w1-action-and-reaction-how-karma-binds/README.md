# C2-W1 — Action and Reaction: How Karma Binds

## Purpose

Introduce karma as the law by which materially motivated actions produce consequences and bind the living being, while showing that work offered for Viṣṇu can be purified.

## Cycle

Cycle 2 — first six months (Year 1 foundation)

## Key verse

Bhagavad-gītā 3.9 — yajñārthāt karmaṇo ’nyatra ... karma-bandhanaḥ

## Minimum home practice

Use the before–during–after method once on a routine duty.

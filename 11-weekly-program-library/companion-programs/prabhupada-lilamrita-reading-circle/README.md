**Launch status:** `planned-after-friday-program-stabilization`

## Format

30 minutes (normally online):

- 2 min — opening
- 20–22 min — shared reading (participant-owned copies)
- 5–6 min — brief realizations (optional)
- 2 min — close and next assignment

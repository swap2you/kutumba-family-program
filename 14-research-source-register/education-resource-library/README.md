# Education Resource Library

Metadata-first maps of official ISKCON education and congregation resources derived from `KUT-SRC-0013`.

## Scope

- ISKCON Ministry of Education pages and media library indexes
- Sunday School curriculum indexes (Madhava / Damodar bands)
- Bhakti-vṛkṣa congregation training modules
- Pattern libraries for KUTUMBA-authored adaptations only

## Not in scope

- Full third-party PDFs, workbooks, or teacher guides in Git
- Replacing KUTUMBA three-year architecture with Bhakti-vṛkṣa sequence
- Restoring deprecated three-band child model

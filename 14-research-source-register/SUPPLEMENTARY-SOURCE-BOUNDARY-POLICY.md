# Supplementary Source Boundary Policy

| Field | Value |
|---|---|
| Version | 1.0.0 |
| Effective | 2026-06-30 |
| Authority | Research Lead + Doctrinal Review (human) |

## Purpose

Define how PureBhakti, Harikatha, Gita Press, and similar sources may be used without weakening the Prabhupāda-first controlling layer.

## Classifications

| Class | Meaning |
|-------|---------|
| `recognized-gaudiya-supplementary` | Gauḍīya Vaiṣṇava materials — exact attribution required |
| `sanatana-narrative-supplementary` | Broader Sanātana narrative/edition discovery |
| `edition-comparison-only` | Compare editions; never merge translations silently |
| `media-discovery-only` | Metadata and links; no bulk download |
| `pending-doctrinal-review` | Not for teaching use until reviewed |

## Rules

1. Prabhupāda's books and Tier A sources remain the normal controlling teaching layer.
2. Supplementary sources do not automatically override ISKCON-controlling doctrine.
3. Exact translator, editor, edition, and publisher must be recorded before use.
4. No quoting controversial positions without context and human review.
5. Public access is not redistribution permission.

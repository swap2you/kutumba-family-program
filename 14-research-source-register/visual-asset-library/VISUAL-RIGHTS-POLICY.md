# Visual Rights Policy (V7)

## Rights classes

| Class | Git commit | Use |
|---|---|---|
| `kutumba-original` | Yes — SVG/PNG | KUTUMBA-authored diagrams |
| `public-domain-verified` | Yes with license record | After verified PD/CC |
| `cc-licensed-verified` | Yes with license record | After verified CC terms |
| `permission-granted` | Yes with permission record | Congregation-owned art |
| `link-reference-only` | Metadata only | BBT art, Krishna.com, photos |
| `permission-required` | No binary | Catalog URL + request register |
| `rights-unclear-do-not-use` | No | Blocked from Gamma/modules |

## Prohibited in Git without clearance

- BBT paintings and book page screenshots
- Prabhupāda photographs
- YouTube thumbnails
- Third-party coloring books
- Copied devotional paintings

Public availability ≠ redistribution permission.

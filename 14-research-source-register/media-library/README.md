# Media Library

Controlled metadata catalog for KUTUMBA curriculum media. **No copyrighted audio/video files in Git by default.**

## Catalogs

| File | Purpose |
|---|---|
| [PRABHUPADA-LECTURE-CATALOG.yaml](PRABHUPADA-LECTURE-CATALOG.yaml) | Verified Śrīla Prabhupāda lectures |
| [APPROVED-TEACHER-CATALOG.yaml](APPROVED-TEACHER-CATALOG.yaml) | Tier-2 teachers (empty until approved) |
| [KATHA-AUDIO-CATALOG.yaml](KATHA-AUDIO-CATALOG.yaml) | Approved katha audio metadata |
| [BHAJANA-AND-KIRTANA-CATALOG.yaml](BHAJANA-AND-KIRTANA-CATALOG.yaml) | Bhajana/kīrtana references |
| [VIDEO-CATALOG.yaml](VIDEO-CATALOG.yaml) | External video metadata |
| [MEDIA-RIGHTS-REGISTER.yaml](MEDIA-RIGHTS-REGISTER.yaml) | Rights status per entry |
| [TOPIC-TO-MEDIA-MAP.md](TOPIC-TO-MEDIA-MAP.md) | Module topic → media index |

## Governance

- Tier 1: Śrīla Prabhupāda — VedaBase stable links
- Tier 2: Approved speakers — `approved-speakers.yaml` (empty)
- No inferred or partial teacher names

## Integration

Each module links via `audio-video/MEDIA-INDEX.yaml`.

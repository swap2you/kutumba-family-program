# Cycle Celebration Addendum

Module IDs (`C1-W1` …) are **stable module identifiers**, not mandatory consecutive calendar weeks.

## Recommended cycle arc

1. Modules 1–5: concept and application
2. Module 6: integration, project completion, rehearsal
3. Protected off week
4. Cycle Utsava / showcase
5. Begin next cycle when families are ready

## Annual rhythm

The three-year architecture controls **40 active weeks per year**. Celebration weeks may align with temple festivals when coordinated — do not hard-code dates.

## Privacy at celebrations

Do not reveal private home-practice records, assessment scores, family struggles, or mentor notes.

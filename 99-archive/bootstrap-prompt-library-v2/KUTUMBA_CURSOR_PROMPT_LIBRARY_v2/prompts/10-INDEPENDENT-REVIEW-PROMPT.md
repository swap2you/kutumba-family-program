# Prompt 10 — Independent Read-Only Review

Run this in a fresh Cursor, Claude Code, Codex, or comparable review context after the build.

---

Audit the KUTUMBA repository in read-only mode. Do not fix files during this review.

Do not trust the builder’s summary without checking source files and Git evidence.

## Required verification

1. Confirm the repository root is not nested.
2. Confirm the remote is private and correct.
3. Verify copied current-source hashes against the manifest.
4. Compare the canonical Markdown with each supplied operating document.
5. Verify the full 18 active sessions remain detailed and complete.
6. Verify missing workstreams are honestly marked rather than fabricated.
7. Verify the three-year architecture and 40-active-week rhythm remain intact.
8. Verify legacy Bhaktivriksha and Granth material is reference-only unless formally adopted.
9. Verify no bulk copyrighted book/PDF collection was committed.
10. Verify scriptural citations are traceable and unverified claims are marked.
11. Verify worship procedures are not invented.
12. Verify child-safety and privacy exclusions are enforced.
13. Verify no real family, child, medical, attendance, incident, mentor, contact, or credential data exists.
14. Verify repository navigation and status reporting.
15. Verify validation scripts actually ran and can be rerun.
16. Verify Git history is understandable.
17. Verify no application, database, website framework, or needless software platform was introduced.
18. Review sample lesson files from all three first-six-month cycles.
19. Review the legacy crosswalk for unjustified adoption.
20. Review the prompt library for consistency and completeness.

## Required report

Create:

`17-reviews-and-audits/INDEPENDENT-REPOSITORY-AUDIT.md`

Use severity:

- Critical
- High
- Medium
- Low

Give one verdict:

- GO
- GO WITH CONDITIONS
- NO GO

For every finding include:

- evidence
- affected path
- impact
- exact remediation
- verification step

Do not provide a GO verdict with unresolved Critical or High findings.
